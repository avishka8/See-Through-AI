# 🧠 See Through AI – Assistive Vision System

## 🚀 Overview
See Through AI is a real-time AI-powered assistive vision system designed to help visually impaired individuals understand their surroundings using a webcam.

It uses computer vision and YOLO-based object detection to identify objects in real time and provide audio feedback.

---

## 🎯 Features
- Real-time object detection using webcam
- YOLO-based object recognition
- Audio feedback using Text-to-Speech
- Multi-object detection support
- Basic distance estimation (if implemented)
- Lightweight and real-time processing

---

## 🛠️ Tech Stack
- Python
- OpenCV
- YOLO (Ultralytics / Custom Model)
- PyTorch / TensorFlow
- NumPy
- pyttsx3 / gTTS
